A simple, free, lightweight notepad & list making app. Made to practice Kotlin, Room, and coroutines. Find it on Google Play [here](https://play.google.com/store/apps/details?id=com.lucasjwilber.freenote)!

* Kotlin
* Kotlin coroutines
